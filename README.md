## Descripción

En este repositorio se encuentran los ejercicios realizados para la Actividad 1 de Programación Web.

El objetivo de estos ejercicios fue practicar el uso de HTML, CSS, JavaScript  creando páginas web con elemntos y herramientas vistos en clases.

## Contenido del repositorio

- **Ejercicio 1:** Hola Mundo.
- **Ejercicio 2:** Párrafos y formato básico.
- **Ejercicio 3:** Enlaces simples.
- **Ejercicio 4:** Imágenes básicas.
- **Ejercicio 5:** Combinando texto y enlaces.
- **Ejercicio 6:** Secciones semánticas.
- **Ejercicio 7:** Listas anidadas.
- **Ejercicio 8:** Horario de clases.
